1. Criando Conta Educacional

Onshape oferece acesso gratuito para estudantes e educadores:

  • Acesse onshape.com/en/education
  • Clique em "Get Started"
  • Selecione "Education Plan"
  • Use email educacional (.edu) se possível
  • Preencha formulário de verificação
  • Aguarde aprovação (1-2 dias)
Vantagens do Onshape:
  • 100% baseado em nuvem (acesse de qualquer lugar)
  • Colaboração em tempo real
  • Versionamento automático
  • Apps mobile (iOS/Android)
  • Não requer hardware potente

2. Interface do Onshape

Elementos principais da interface:

  • Toolbar: Ferramentas de modelagem (topo)
  • Feature List: Histórico de operações (esquerda)
  • Viewport: Área de visualização 3D (centro)
  • ViewCube: Navegação de vista (canto)
  • Parts List: Componentes do documento (baixo)
Navegação 3D: Mouse Scroll - Zoom Botão Meio - Pan (arrastar) Botão Direito - Rotacionar Shift + Botão Meio - Rotacionar Atalhos: S - Sketch (esboço) E - Extrude R - Revolve F - Fillet (arredondar) C - Chamfer (chanfro) M - Mirror (espelhar)

3. Primeiro Projeto: Cubo Simples

Crie sua primeira peça 3D:

  • Clique em "Create" → "Document"
  • Nomeie o documento
  • Clique em "Part Studio" (já aberto)
  • Clique em "Sketch" (S)
  • Selecione plano "Top"
  • Desenhe retângulo 50x50mm
  • Clique em ✓ (check) para finalizar sketch
  • Clique em "Extrude" (E)
  • Defina altura: 50mm
  • Clique em ✓
Parabéns! Você criou seu primeiro sólido 3D no Onshape!

4. Ferramentas de Sketch

Desenho 2D para criar perfis:

Ferramentas Básicas: L - Line (linha) C - Circle (círculo) R - Rectangle (retângulo) A - Arc (arco) P - Point (ponto) T - Text (texto) Restrições (Constraints): H - Horizontal V - Vertical = - Equal (igualdade) ⊥ - Perpendicular ∥ - Parallel ⌀ - Dimension (cota) Dica: Use restrições para definir geometria completamente (fully defined)

5. Features 3D Principais

Operações para criar sólidos:

  • Extrude: Extruir perfil 2D em 3D
  • Revolve: Girar perfil em torno de eixo
  • Sweep: Seguir caminho
  • Loft: Transição entre perfis
  • Fillet: Arredondar arestas
  • Chamfer: Chanfrar arestas
  • Shell: Criar casca (oco)
  • Pattern: Repetir features
Exemplo: Cilindro Oco 1. Sketch → Circle ⌀50mm 2. Extrude → 100mm 3. Shell → Espessura 5mm (selecione face superior) Resultado: Tubo com parede de 5mm

6. Assemblies (Montagens)

Combine múltiplas peças:

  • Clique em "+" → "Create Assembly"
  • Insira peças: "Insert" → Selecione Part Studio
  • Use "Mate" para posicionar peças:
    • Fastened: Fixar posição
    • Revolute: Rotação (dobradiça)
    • Slider: Deslizamento
    • Cylindrical: Rotação + deslizamento
  • Teste movimento com "Animate"
Exemplo: Monte um robô com braços articulados usando mates Revolute nas juntas.

7. Drawings (Desenhos Técnicos)

Crie documentação técnica:

  • Clique em "+" → "Create Drawing"
  • Selecione template (A4, A3, etc.)
  • Arraste vistas da peça
  • Adicione cotas (dimensions)
  • Insira notas e tolerâncias
  • Exporte como PDF ou DWG
Vistas Padrão: - Front (frontal) - Top (superior) - Right (lateral direita) - Isometric (isométrica) - Section (corte) - Detail (detalhe ampliado)

8. Colaboração em Tempo Real

Trabalhe em equipe simultaneamente:

  • Clique em "Share" (canto superior direito)
  • Adicione colaboradores por email
  • Defina permissões:
    • View: Apenas visualizar
    • Comment: Visualizar e comentar
    • Edit: Editar documento
    • Owner: Controle total
  • Veja cursores de outros usuários em tempo real
  • Use "Follow" para acompanhar trabalho de colega

9. Versionamento e Histórico

Controle de versões automático:

  • Cada edição é salva automaticamente
  • Crie "Versions" para marcos importantes
  • Compare versões lado a lado
  • Restaure versões anteriores
  • Veja histórico completo de mudanças
Criar Versão: 1. Clique em ícone de relógio (History) 2. Clique em "Create version" 3. Nomeie (ex: "v1.0 - Protótipo inicial") 4. Adicione descrição 5. Clique em "Create" Benefício: Nunca perca trabalho!

10. Projeto Prático: Suporte para Arduino

Especificações: - Base: 70mm x 55mm x 3mm - Furos M3 nas posições do Arduino Uno - Espaçadores: 10mm altura - Encaixe para cabo USB Passos: 1. Sketch retângulo 70x55mm 2. Extrude 3mm (base) 3. Adicione círculos ⌀3.2mm (furos M3) Posições: (14,2.5), (66,7.5), (66,35.5), (14,50.5) 4. Extrude Remove (furos passantes) 5. Sketch círculos ⌀6mm nos mesmos pontos 6. Extrude 10mm (espaçadores) 7. Fillet 1mm nas arestas 8. Adicione recorte para USB Exportar para impressão 3D: - Clique direito na peça - Export → STL - Imprima em PLA ou ABS
