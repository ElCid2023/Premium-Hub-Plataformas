⚙️ Tutorial Arduino IDE 2.0

Ambiente Oficial de Desenvolvimento Arduino

1. Download e Instalação

  • Acesse arduino.cc/software
  • Baixe Arduino IDE 2.0
  • Instale no seu sistema
  • Conecte Arduino via USB

2. Configuração

Selecione placa e porta:

  • Tools → Board → Arduino Uno
  • Tools → Port → COM3 (Windows) ou /dev/ttyUSB0 (Linux)

3. Primeiro Código

void setup() { pinMode(LED_BUILTIN, OUTPUT); } void loop() { digitalWrite(LED_BUILTIN, HIGH); delay(1000); digitalWrite(LED_BUILTIN, LOW); delay(1000); }

4. Upload

Clique no botão Upload (→) ou Ctrl+U

5. Monitor Serial

Tools → Serial Monitor (Ctrl+Shift+M)

Configure baud rate: 9600
