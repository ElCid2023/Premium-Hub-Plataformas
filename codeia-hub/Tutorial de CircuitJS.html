⚡ Tutorial CircuitJS

Simulador de Circuitos Online

1. Acesso

Acesse falstad.com/circuit/circuitjs.html

Não requer instalação ou cadastro

2. Adicionar Componentes

  • Clique direito na área de trabalho
  • Selecione componente (resistor, LED, etc)
  • Clique para posicionar
  • Arraste para conectar

3. Simulação em Tempo Real

Visualização instantânea:

  • Verde: Tensão positiva
  • Cinza: Zero volts
  • Vermelho: Tensão negativa
  • Pontos amarelos: Corrente

4. Osciloscópio

Visualizar sinais: 1. Clique direito no fio 2. Selecione "View in Scope" 3. Ajuste escala de tempo 4. Compare múltiplos sinais

5. Salvar Circuito

  • File → Export as Link (URL)
  • File → Export as Text
  • Copie e salve o código
